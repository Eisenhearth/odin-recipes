
A simple HTML recipe project built as a beginner-friendly web page. This repository contains a single static recipe page that can be opened directly in the browser.

## Project Overview

- **Framework:** Plain HTML
- **Entry point:** `index.html`
- **Purpose:** Serve a recipe page layout with structured content for ingredients, instructions, and preparation details.

## How to Use

1. Open `index.html` in your browser.
2. Edit the file to add your own recipe content.
3. Save and refresh the browser to view updates.

## Project Structure

- `index.html` – main HTML page for the recipe content
- `README.md` – project documentation
